# TechMais Store Inventory System

## About the Project

This project was developed as part of the course **Python: Data Persistence with Files, Databases, and REST APIs**.

The goal of this activity is to practice data persistence using **Python** and **SQLite**, creating a simple inventory control system for a fictional store called **TechMais**.

---

## Activity Context

TechMais Store is improving its inventory management process and needs a simple Python system to register and list products stored in a database.

The system must:

- Connect to a SQLite database;
- Create a products table;
- Insert three different products;
- List all registered products.

---

## Technologies Used

| Technology | Purpose                                                       |
| ---------- | ------------------------------------------------------------- |
| Python     | Main programming language used to build the system            |
| SQLite     | Lightweight relational database used to store product data    |
| sqlite3    | Native Python module used to connect and interact with SQLite |
